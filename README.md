# 📊 Customer Churn Prediction System

## 🚀 Overview

A Machine Learning project that predicts whether a customer will churn based on service usage and payment details.

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

---

## 📁 Files

* `churn_project.ipynb` → Model training + prediction
* `churn.csv` → Dataset

---

## ⚙️ Workflow

1. Data preprocessing
2. Feature selection
3. Model training (Random Forest)
4. Evaluation
5. Prediction

---

## 📊 Model

* Algorithm: Random Forest
* Accuracy: ~75–85%

---

## ▶️ Run

Open and run:

```bash
churn_project.ipynb
```

---

## 💡 Features

* Predicts customer churn
* Displays confidence score
* Simple input-based prediction
