# 🚀 Machine Learning Projects

This repository contains two machine learning projects focused on real-world applications using data analysis and NLP.

---

## 🌾 Crop Yield Recommendation System

### 🚀 Overview

A system that recommends the most suitable crop based on environmental and soil conditions to maximize yield.

### 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn

### ⚙️ Features

* Takes input: soil type, rainfall, temperature, humidity
* Predicts best crop for cultivation
* Data preprocessing and model training

---

## 📧 Email Spam Classifier

### 🚀 Overview

A machine learning model that classifies messages as **Spam** or **Not Spam (Ham)** using NLP techniques.

### 🧠 Tech Stack

* Python
* Scikit-learn
* TF-IDF (n-grams)
* Naive Bayes

### ⚙️ Features

* Text preprocessing and cleaning
* TF-IDF vectorization
* Spam detection with high accuracy (~95–98%)
* Explainable output (top spam keywords)
* Confidence score for predictions


## 📊 Applications

* Agriculture decision support
* Email/message filtering systems
